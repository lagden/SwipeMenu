It's a lightweight JavaScript library for swiping off-canvas menus


## Install

Via [NPM](https://www.npmjs.com/)

```
npm i -S github:felipoliveira/SwipeMenu
```


## Usage

```javascript
const swipeMenu = SwipeMenu('#menu');
```
